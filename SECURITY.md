# Security

This project is a local Codex skill and glossary tool. It does not need API keys, passwords, or cloud credentials.

## Reporting

If you find a security issue, please open a GitHub issue with a minimal reproduction and avoid posting secrets or private data.

## Data Safety

- Do not commit private user conversations.
- Do not commit API keys, tokens, passwords, or local machine paths.
- Keep glossary entries focused on public technical vocabulary.
